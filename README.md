# keyplane

A minimal TypeScript starter scaffolded with **devkit**.

## Scripts

- `npm run dev` – Run the source directly with tsx.
- `npm run build` – Bundle to `dist` via tsup (ESM + CJS + types).
- `npm run test` – Execute unit tests with Vitest.
- `npm run start` – Run the bundled output.

## Development

1. Edit code in `src/`.
2. Run `npm run dev` while iterating, or `npm run build` to produce distributable artifacts.
3. Publish to npm after building; `dist/` and type declarations are generated by tsup.
